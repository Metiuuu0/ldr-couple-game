# ❤️ Whenever You Miss Me

A small personal website created as a gift for someone special.

**Whenever You Miss Me** is a private collection of voice messages, designed to make distance feel a little smaller. Whenever you miss someone, you can open the page and listen to a message from them with just one click.

## ✨ Features

* 💙 Personalized voice recordings
* 🎵 Built-in audio player
* 📱 Mobile-friendly design
* 💕 Simple and romantic interface
* 🌸 Custom photos and messages
